# Documentation Index

This folder contains the delivery-ready documentation package for the `asset_managemt_final` monorepo. It is written as a realistic handoff bundle for a teammate, reviewer, lecturer, or deployment owner.

## Core Product and Requirements

| File | Focus |
| --- | --- |
| `software-overview.md` | Business context, goals, scope, assumptions, constraints, and non-functional requirements |
| `use-case-specification.md` | Detailed use cases with actors, preconditions, flows, validations, authorization rules, and data effects |
| `functional-flows.md` | Cross-module workflow explanations with activity and sequence diagrams |

## Architecture and Engineering Design

| File | Focus |

| `api-overview.md` | API surface, endpoint groups, and error model |
| `rbac-design.md` | Role model, grant matrix, enforcement layers, and search/RBAC interaction |
